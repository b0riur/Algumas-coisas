Adiciona 2 atalhos de seleção no menu de contexto da Área de Trabalho.
1 Sendo para limpar os arquivos temporários do Windows em "%temp%" e "temp".

2 Sendo para reiniciar o Windows Explorer, ideal caso surja algum erro ou travamento no Windows.

*Adicione o arquivo 'Limpar-Temp.ps1' em C:\Windows, ficará assim: "C:\Windows\Limpar-Temp.ps1"*
